  VirtualCharacters, 
  WebGLEffects,
  initThreeJsGlobally

// Initialize Three.js globally

// Create a character controller instance
const characterController = new CharacterController(model, {
  scale: 1,
  position: { x: 0, y: 0, z: 0 },
  idleAnimation: 'idle'
});

Method 4 provides a clean, simple way to import just what you need from the character system. It uses the convenience import files we created in src/threeJs/ that re-export specific components.

This approach gives you maximum flexibility while maintaining clean import statements in your code. You can import just the specific utility or component you need without having to remember the full path to the source file.
