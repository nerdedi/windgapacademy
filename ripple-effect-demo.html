// Create a ripple effect for a button
const buttonRipple = createCanvasRippleEffect(canvas, {
  color: '#ffffff',
  rippleOpacity: 0.6,
  rippleSpeed: 2,
  maxRadius: 100,
  lineWidth: 2,
  fadeSpeed: 0.96
});

// Add a ripple when the button is clicked
button.addEventListener('click', (e) => {
  const rect = button.getBoundingClientRect();
  const x = e.clientX - rect.left;
  const y = e.clientY - rect.top;
  buttonRipple.addRipple(x, y);
});

Implementation Guide

Follow these steps to integrate ripple effects into your Windgap Academy UI components:

  1. Import the required utilities: 
    import { createRippleEffect, createCanvasRippleEffect, createInteractiveRippleBackground } from '../utils/RippleEffectUtils';
  2. For React components, use the RippleEffect wrapper: 
    import RippleEffect from '../components/RippleEffect';

// In your component:
<RippleEffect color="#4f46e5" duration={800} rippleOpacity={0.3}>
  <button>Click me</button>
</RippleEffect>
  3. For vanilla JS implementation, create an interactive background: 
    const container = document.querySelector('#my-container');
const rippleEffect = createInteractiveRippleBackground(container, {
  rippleColors: ['rgba(79, 70, 229, 0.3)'],
  autoRippleInterval: 3000
});
